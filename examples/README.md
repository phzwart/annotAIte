# Examples

This directory contains example notebooks demonstrating how to use annotAIte.

## Usage

1. Start with `test_pipeline_basic.ipynb` for a simple end-to-end workflow example
2. Load an image, run the pipeline, and visualize results
3. Experiment with different parameters (patch_size, k, etc.)

## Running Examples

Make sure you have annotAIte installed in development mode:

```bash
pip install -e ".[napari]"
```

Then launch Jupyter:

```bash
jupyter notebook
# or
jupyter lab
```
